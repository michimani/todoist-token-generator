todoist OAuth Authorization

Steps
  1. Create todoist app
  2. The authorization request
  3. The redirection to your application site
  4. The token exchange

Step 0: Create todoist app

Access to Todoist App Management Console and create a new todoist app.
Then, please input following url to OAuth redirect URL form.

http://127.0.0.1:8888/auth/step2/

Create auth.json file

Please create auth.json file with following format, and put it to /auth/ directory. 

    {
      "client_id": "your_client_id",
      "client_sec": "your_client_secret",
      "state": "random_string_you_like"
    }
